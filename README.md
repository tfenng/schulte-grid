# Schulte Grid

Windows desktop MVP for Schulte Grid training, built with Tauri, React, TypeScript, and Vite.

## Commands

- `npm install`
- `npm run tauri dev`
- `npm test`
- `npm run build`
- `cargo check --manifest-path src-tauri/Cargo.toml`

## MVP Scope

- Home page with best score and history entry
- Mode selection for `3 x 3`, `4 x 4`, and `5 x 5`
- Ascending-only training board with pause/resume
- Result page and persistent local history
